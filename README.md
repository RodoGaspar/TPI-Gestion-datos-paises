# TPI-Gestion-datos-paises

## Descripción
Trabajo Práctico Integrador para la materia Programación 1 de la Tecnicatura Universitaria en Programación de la Universidad Tecnológica Nacional. Está desarrolado en python y funciona cómo un sistema de gestión de datos de países.

## Requisitos
Para correr el programa es necesario tener instalado Python 3.x
Se puede verificar la instalación ejecutando en la terminal

```bash
python --version
```

## Instrucciones uso
1. Clonar el proyecto desde el repositorio

 ```bash
 git clone https://github.com/RodoGaspar/TPI-Gestion-datos-paises.git
 ```

2. Dirigirse a la carpeta del proyecto ejecutando en la terminal:

```bash
cd TPI-Gestion-datos-paises
```

3. Correr el programa ejecutando en la terminal

```bash
python paises.py
```
